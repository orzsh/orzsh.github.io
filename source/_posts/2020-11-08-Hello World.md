折腾了一天多，这个blog终于搭建好了，本次采用Hexo强力驱动，挂载在GitHub上，Theme是[Yilia](https://github.com/litten/hexo-theme-yilia)（感谢@Litten大佬将自己的Theme源码贡献出来）。很早之前就有搭建个人blog的想法，但由于许多原因最终一拖再拖至今天。说是blog，但我更愿意称之为网络记事本。

写blog的目的是为了记录自己学习的过程，心理上的问题，生活中的琐事，以及一些无用的思考，以此督促自己不断向前。

这片狭小的空间，它不像QQ空间，朋友圈，知乎，它更像是属于自己的一块小小领地，我能在这里畅所欲言，随心挥洒，并满足于这种归属感。

## Quick Start

### Create a new post

``` bash
$ hexo new "My New Post"
```

More info: [Writing](https://hexo.io/docs/writing.html)

### Run server

``` bash
$ hexo server
```

More info: [Server](https://hexo.io/docs/server.html)

### Generate static files

``` bash
$ hexo generate
```

More info: [Generating](https://hexo.io/docs/generating.html)

### Deploy to remote sites

``` bash
$ hexo deploy
```

More info: [Deployment](https://hexo.io/docs/one-command-deployment.html)
